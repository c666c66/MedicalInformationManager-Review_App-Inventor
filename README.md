## 製作動機 ##
畢業專題小組成員皆有報考醫學資訊管理師，為了方便複習考古題，便將畢業專題題目定為醫學資訊管理師考古題複習系統。

## 關於 APP ##
為了讓醫學資訊管理師的考生們有效的複習考古題，「醫學資訊管理師考古題複習 APP」所具備的功能：  
1. 歷屆考古題瀏覽。  
2. 《常用醫護術語》考古題語音朗讀、中文翻譯。  
3. 模擬考。
